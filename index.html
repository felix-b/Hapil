Happil

A Reflection.Emit wrapper for dynamic implementation of interfaces and abstract classes, which just works and keeps its users happy.

Download as .zip Download as .tar.gz View on GitHub

Welcome to Happil.

This project is aimed to create a Reflection.Emit wrapper library, one that would precisely fit use cases of runtime type generation. We launched this project because we feel that existing alternatives don't fit that well.

Use Cases

  • Implementation: developers define interfaces, then you have to provide implementations of the interfaces at runtime.
  • Interception: you need to decorate objects with implementation of cross-cutting concerns.

Alternatives

  • RunSharp
  • FluentIL
  • Castle DynamicProxy
